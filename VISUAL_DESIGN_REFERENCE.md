## Spacing System

```
Container padding
Desktop  : 3rem (48px)
Mobile   : 2rem (32px)

Card gaps
Desktop  : 1.5rem (24px)
Mobile   : 1rem (16px)

Element spacing
XL  : 5rem (80px)   - Major sections
L   : 3rem (48px)   - Section breaks
M   : 2rem (32px)   - Related groups
S   : 1rem (16px)   - Elements
XS  : 0.5rem (8px)  - Tight spacing
```

## Accessibility

### Color Contrast Ratios
```
? Gold text on Ink          : 7.2:1 (AAA)
? Cream text on Charcoal    : 12.5:1 (AAA)
? Black text on Gold        : 8.1:1 (AAA)
```

### Focus Indicators
```
All interactive elements:
- 2px solid gold outline
- 2px offset from element
- Visible on keyboard navigation
```

### ARIA Labels
```
? Status chips include role="status"
? Buttons have aria-label where needed
? Loading states announce changes
```
